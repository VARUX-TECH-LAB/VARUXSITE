Yeni Nesil Çözümler

5 proje listeleniyor

VARUX INTEL

Açık kaynak istihbarat (OSINT) sistemimiz. Dark web forumları, sızıntı siteleri ve Telegram kanallarını tarayarak markaya veya kuruma dair veri ihlali sinyallerini tespit eder.

Tamamlanma: Temmuz 2024
Süre: 12 Hafta
Teknoloji: Python, Scrapy, Elasticsearch, BeautifulSoup
Threat Intelligence OSINT Dark Web Monitoring

VARUX TRAPNET

Gerçek saldırgan davranışlarını yakalamak için dağıtılmış sahte sistemlerden oluşan bir ağ. Web, SSH ve FTP gibi servisleri taklit ederek saldırı vektörlerini kaydeder, analiz eder.

Tamamlanma: Haziran 2024
Süre: 10 Hafta
Teknoloji: Cowrie, Dionaea, Docker, Kibana
Honeypot Ağ Güvenliği Saldırı Analizi

VARUX ANALYZER

Zararlı yazılımların davranışlarını izole sanal ortamda analiz eder. Dosya çalıştırıldığında API çağrıları, dosya değişiklikleri ve ağ trafiği izlenir. Sonuçlar JSON veya HTML rapor olarak üretilir.

Tamamlanma: Mayıs 2024
Süre: 14 Hafta
Teknoloji: Cuckoo Sandbox, Python API Wrapper
Malware Analysis Sandbox Davranış Analizi

VARUX PHISHGUARD

Gelen e-postalarda oltalama saldırılarını tespit eden makine öğrenimi tabanlı sistem. E-posta başlıklarını, içerikleri ve bağlantıları analiz eder. Kurumsal e-posta altyapısına entegre edilerek gerçek zamanlı koruma sağlar.

Tamamlanma: Nisan 2024
Süre: 8 Hafta
Teknoloji: scikit-learn, Flask API, IMAP, DNS Resolver
Phishing Detection E-posta Güvenliği Makine Öğrenimi

VARUX DEEPFLOW

Şifreli trafiği çözmeden anomali ve sızma girişimlerini tespit eden sistem. TLS handshake ve paket istatistiklerinden davranışsal imzalar çıkarır. Zero-day saldırılarını dahi tespit edebilecek derin öğrenme tabanlı modelle çalışır.

Tamamlanma: Mart 2024
Süre: 16 Hafta
Teknoloji: Wireshark libs, TensorFlow, Python
Ağ Güvenliği Traffic Analysis Derin Öğrenme
